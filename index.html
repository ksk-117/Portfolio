〇Prime Number Poker


制作時期:2024年1月

このゲームは5つの数字を好きな順番に並べ替えて、
素数を作って高得点を目指すゲームです。

[例]（本当は5桁です）
253 → 2, 3, 5, 53 … 1100pt
↓↓並び替えて↓↓
523 → 2, 3, 5, 23, 523 … 2100pt
このように、より高得点を目指していきます。

カードの入れ替える方法は
左クリックで1回目にクリックしたカードと
2回目にクリックしたカードを交換することができます。
また、右クリックで新しい数字に変更できます。
様々な手役を作り高得点を目指していこう！

また、Githubリポジトリに同封している「judge.py」は
「prime.py」の補助ツール(チートツール)として作成したもので
任意の数字の羅列を入力し、
それを並び替えてできる素数をすべて出力するプログラムです。

＜＜詳しくはこちら＞＞

〇PDFのリサイズツール

制作時期:2024年2月 (12時間以上)


これは複数の画像ファイルを均一の大きさのPDFに変換するためのツールです。
主にテキストやプリントをスキャンしてPDFを作成し書き込んで学習する場合や、
書類をスキャンしてPDFを先生に提出する場合に利用します。

～使用方法～

1, ダウンロードしパッケージをインストールする。
2, fixed_size を実行します。
3, "画像フォルダを選択" ボタンをクリックして、変換したい画像が含まれるフォルダを選択します。
4, "A4縦向きに出力" または "A4横向きに出力" ボタンをクリックして、PDFに変換します。
5, 変換が完了すると、"output.pdf"がダウンロードされます。
6, 作成されたPDFファイルがプレビューされます。
(テストプレイ用に"example"フォルダ(jpg画像複数枚)を用意しています。)

次のようにサイズが変更できます。
